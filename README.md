# Bootstrap Portfolio

Responsive Bootstrap 5 portfolio for the "Clone Your Future" assignment.

## Implemented sections

- Full-bleed hero with centered text, dark overlay, sticky navbar, active underline, and skip-to-content link.
- Two-column About Me card with profile details, Bootstrap skill badges, highlight label, and concise bio under 160 words.
- Seven-project responsive grid using the actual linked projects in the portfolio:
  - HistoQuiz App: live site and GitHub repository.
  - Tesla Landing Page: GitHub repository.
  - Netflip Clone: live site and GitHub repository.
  - MeTube Clone: GitHub repository.
  - Chef Menu App: Expo Snack demo and GitHub repository.
  - KinderNest: live site and GitHub repository.
  - To Do List App: live site and GitHub repository.
- Contact section with Bootstrap Icons, accessible labels, required fields, email validation, and hover/focus states.
- Dark footer with the required attribution.

## Bonus

- Print-ready resume view using `@media print`.
- Auto dark mode using `prefers-color-scheme` and CSS variables.

## Image credits

Hero and profile visuals in `assets/` are original SVG assets created for this assignment. Project card images use remote screenshots from the linked live/demo project pages and GitHub repository preview images, so the grid reflects the actual projects shown in the portfolio.
