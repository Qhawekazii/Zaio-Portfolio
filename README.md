# Bootstrap Portfolio

Responsive Bootstrap 5 portfolio for the "Clone Your Future" assignment.

## Implemented sections

- Full-bleed hero with centered text, dark overlay, sticky navbar, active underline, and skip-to-content link.
- Two-column About Me card with profile details, Bootstrap skill badges, highlight label, and concise bio under 160 words.
- Six-project responsive grid: Tesla, Netflix, YouTube, Weather Dashboard, Restaurant Website, and Personal Portfolio.
- Contact section with Bootstrap Icons, accessible labels, required fields, email validation, and hover/focus states.
- Dark footer with the required attribution.

## Bonus

- Print-ready resume view using `@media print`.
- Auto dark mode using `prefers-color-scheme` and CSS variables.
